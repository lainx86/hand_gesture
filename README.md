# Hand Gesture Recognition

This project uses computer vision to recognize hand gestures from an image.

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/hand-gesture-recognition.git
    ```
2.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

Run the `main.py` file to start the gesture recognition:

```bash
python main.py
```

## Files

*   `main.py`: The main script to run the application.
*   `gesture_detector.py`: Contains the `GestureDetector` class for hand gesture recognition.
*   `image_utils.py`: Contains utility functions for image processing.
*   `assets/`: Directory containing images for the application.
